Documentation and capabilities of the build and starter template

Launching in development mode: gulp
Building optimized version for production: gulp docs

Starter template capabilities

  • Ready-made styles for CSS containers for all life situations.
  • Mixins and snippets for responsiveness.
  • Mixin and snippet for CSS retina background images.

Build capabilities

  • Auto-correction of paths. Feel free to use editor hints when specifying file paths. The build will automatically remove unnecessary upward references ../../ in src HTML and background-image in SCSS.
  • Automatic generation of SVG sprites. See details below.

Production build features with gulp docs

  • Automatic compression of jpg images
  • Automatic generation of webp graphics and specifying webp format in HTML.
  • When specifying the img tag, it's not necessary to include srcset with 2x. The 'gulp-webp-retina-html' plugin in the 'html:dev' task automatically adds the picture tag and source for 2x images. Make sure to save images in @2x resolution.
