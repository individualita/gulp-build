Font auto-generation and CSS integration

  1. Place TTF and OTF fonts in src/fonts/
  2. During build, OTF is converted to TTF inside src/fonts/.
  3. Then TTF is converted to WOFF and WOFF2 from src/fonts/ to build/fonts/.
  4. CSS inclusion styles are generated inside src/scss/base/_fontsAutoGen.scss.
    Important! This file is generated and overwritten automatically.
