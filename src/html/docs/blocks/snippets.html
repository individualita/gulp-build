Useful snippets

Snippets are stored in the .vscode folder in the webCademy.code-snippets file

SCSS snippets for media queries

Usage code: tablet

@include tablet { ... }

Usage code: mobile

@include mobile { ... }

Retina background-image via media query. Usage code: mediaBg

@include mediaBg() {
    background-image: url('./../img/example@2x.jpg');
}

Retina background-image via image-set. Usage code: imgSet

background-image: image-set(
    url('./../img/bg.jpg') 1x,
    url('./../img/bg@2x.jpg') 2x);

HTML snippet for SVG icons from sprite

Usage code: svgIcon

<svg class="icon icon--heart-line">
    <use href="./img/svgsprite/sprite.symbol.svg#heart-line"></use>
</svg>
